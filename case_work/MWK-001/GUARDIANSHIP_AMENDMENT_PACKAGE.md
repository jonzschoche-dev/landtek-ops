## 0. What is NOT changing (and why) — the fee strategy

**The narrow property schedule (only TCT T-32911, ~₱5M) is INTENTIONAL — do NOT expand it.** PH docket
fees scale with the value of the estate placed in issue; scheduling the whole ~₱90M+ estate would make the
filing prohibitively expensive. The whole estate is captured **after appointment**, via the guardian's
**Rule 96 §1 duty to file an inventory of all the ward's estate within 3 months** — not by enlarging the
fee-scaled petition schedule. (See §6.) So the fixes below are about the **ground, the recitals, and the
theory** — never the schedule.

**One caveat to confirm with Botor:** whether an additional docket/filing fee attaches at the inventory
stage (on the value inventoried). If so, budget for it as a post-appointment cost, not a pre-filing one.
